## ✨ About WellNest

**WellNest** is a personal wellness and habit journal built to help you reconnect with yourself — one breath, one habit, one reflection at a time.

Born from a love of growth, healing, and tiny beginnings, **WellNest** offers a gentle home for tracking your emotional, physical, and personal journey — without judgment, pressure, or noise.

> "Growth doesn’t have to be loud to be powerful."

At its heart, **WellNest** is about:
- 🌱 Cultivating small, meaningful habits
- 🌲 Grounding yourself in daily reflections
- 🪺 Building a life you feel safe inside

---

## 🛠️ Built With

- **Front-End:** Django Templates, HTML, CSS (with a forest green styling palette 🌿)
- **Back-End:** Django (Python)
- **Database:** PostgreSQL
- **Hosting:** Render
- **Other Tools:** Trello (for planning), Figma (for design mockups)

---

## 🎯 Features

- ✅ User authentication (sign up, login, logout)
- ✅ Daily Check-In tracking
- ✅ Habit Tracker with CRUD (Create, Read, Update, Delete)
- ✅ Personal Reflection journal entries
- ✅ Emotional Mood Check linked to habit recommendations
- ✅ Forest-themed styling with calming UI/UX

> Future goals: AI-generated wellness reflections 🌿🧠, mood-to-habit visual charts 📈

---

## 🚀 Getting Started

Follow these steps to run WellNest locally:

### Prerequisites

- Python 3.11 installed
- PostgreSQL installed
- Git installed
- (Recommended) Virtual Environment

### Installation

1. Clone the repo

   ```bash
   git clone https://github.com/your-username/wellnest.git
   cd wellnest
   ```

2. Create and activate a virtual environment

   ```bash
   python3 -m venv env
   source env/bin/activate
   ```

3. Install dependencies

   ```bash
   pip install -r requirements.txt
   ```

4. Set up environment variables  
   Create a `.env` file based on `.env.example`

5. Apply migrations

   ```bash
   python3 manage.py migrate
   ```

6. Run the server

   ```bash
   python3 manage.py runserver
   ```

7. Open your browser to:

   ```
   http://localhost:8000
   ```
